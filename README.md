# Bokoa


## Android Brain-Teaser Game

is published on the google play store : 

https://play.google.com/store/apps/details?id=com.slykos.bokoa

## Technical details

This repo have two parts : 
	Level Generator : Generate levels on a Desktop PC in python, so android device does not have to
	Android App : Androdi studio project that use the levels (in .json) produced by the Level Generator
